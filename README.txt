Интерактивный сайт по ДТП Республики Саха (Якутия), 2017–2024 годы

Состав сайта:
- index.html — таблица с фильтрами по столбцам;
- dashboard.html — дашборд с интерактивной картой, KPI-карточками и тремя компактными графиками: по годам, месяцам и дням недели;
- style.css — общие стили сайта;
- dashboard.css — стили дашборда;
- script.js — логика таблицы;
- dashboard.js — логика карты, фильтров, карточек и графиков;
- data_columns.js — названия столбцов;
- data_rows_01.js ... data_rows_04.js — строки таблицы, разбитые на несколько файлов для загрузки на GitHub.

Источник данных:
Таблица составлена на основе открытых данных «Показатели состояния дорожного движения» ГИБДД МВД РФ.

Как открыть локально:
1. Распакуйте архив.
2. Откройте index.html для просмотра таблицы.
3. Откройте dashboard.html для просмотра карты и дашборда.

Важно:
- Для отображения карты и графиков нужен доступ к интернету, потому что используются внешние библиотеки Leaflet, Leaflet.markercluster, Chart.js и картографическая подложка OpenStreetMap.
- Дашборд считает сводные показатели по уникальным ДТП. Это сделано для того, чтобы одно ДТП, представленное несколькими строками участников или транспортных средств, не задваивало погибших, раненых и участников.
- На карте отображаются только ДТП с корректными координатами в пределах Республики Саха (Якутия). Ошибочные координаты вроде 0;0 или явно некорректные значения исключаются из карты, но остаются в сводных расчетах.

Как опубликовать на GitHub Pages:
1. Создайте публичный репозиторий на GitHub.
2. Загрузите все файлы из папки interactive_table_site в корень репозитория.
3. Откройте Settings → Pages.
4. В Source выберите Deploy from a branch.
5. В Branch выберите main и /root.
6. Нажмите Save.
7. После публикации сайт будет доступен по ссылке вида https://ваш-логин.github.io/название-репозитория/.

Как убрать сайт:
Repository → Settings → Pages → Unpublish site.
