ТАБЛИЦА ДТП РЕСПУБЛИКИ САХА (ЯКУТИЯ) ЗА 2017–2024 ГОДЫ

Состав архива:
- index.html — главная страница сайта;
- style.css — оформление;
- script.js — логика поиска, сортировки, фильтров и CSV-экспорта;
- data_columns.js — названия столбцов;
- data_rows_01.js, data_rows_02.js, data_rows_03.js, data_rows_04.js — данные таблицы, разбитые на части для загрузки на GitHub.

Размер таблицы:
- строк с данными: 16605;
- столбцов: 50.

Как открыть локально:
1. Распакуйте архив.
2. Откройте index.html в браузере.

Как опубликовать по ссылке через Netlify:
1. Зайдите на app.netlify.com.
2. Выберите Add new site / Deploy manually.
3. Перетащите распакованную папку interactive_table_site в окно загрузки.
4. Netlify выдаст публичную ссылку на сайт.

Как опубликовать через GitHub Pages:
1. Создайте новый репозиторий GitHub.
2. Загрузите в него файлы index.html, style.css, script.js, data_columns.js и все файлы data_rows_XX.js.
3. Откройте Settings → Pages.
4. В разделе Build and deployment выберите Deploy from a branch, ветку main и папку /root.
5. После публикации GitHub покажет ссылку на сайт.

Важно: после публикации по ссылке данные таблицы становятся публично доступными.


Примечание: данные разбиты на несколько файлов data_rows_XX.js, чтобы каждый файл был меньше ограничения загрузки GitHub через браузер. Загружайте все файлы в репозиторий вместе с index.html, style.css и script.js.
